# Store Mode (SillyTavern Extension)

Story-driven tools for SillyTavern chats: Story Arcs, Author Mimicry, Scenario Blueprints, and optional extras.

## Install
- In SillyTavern: Extensions → Install Extension → paste this repo URL.

## Features
- Story Arcs (genre/tone/pacing guidance)
- Author Mimicry (style guidance)
- Scenario Blueprints (multi-scene beat tracking)
- Extras: summaries, epilogues, “What’s Next”

## Notes
- Story Arcs and Author Styles seed from `data/story_arcs.json` and `data/author_styles.json` on first run.
- All features are optional; you can enable only what you want.
